# Asimov-flix

Asimov Flix is a sci-fi themed video library built using React, Express , MirageJS, HTML and CSS.

## Features Present

- Authentication
  - Login
  - Logout
  - Signup
- Video Listing
- Filter Products by Category
- Sort videos by date
- Playlist Management
- Likes / Dislikes
- History
- Watch later
- Search by video title
- Single Video Page
- Notes on a video
- Loading & Alerts
- Custom Video View Count
